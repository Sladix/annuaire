# Projet de demo Annuaire

## À propos
- Application Front End avec [Preact](https://preactjs.com/) et crée avec [Peact-CLI](https://github.com/developit/preact-cli)
- Création de composants controllés
- [Axios](https://github.com/axios/axios) pour le XHR
- [preact-markdown](https://github.com/laggingreflex/preact-markdown) pour cette page
- [less](http://lesscss.org) pour la mise en forme
- Réalisation d'une API simple avec [Symphony 2](https://symfony.com/doc/2.8/) et crée avec [Symfony Rest Edition](https://github.com/gimler/symfony-rest-edition)
    - [Documentation de l'API](http://localhost:8000/api/doc)

## Pistes d'améliorations
- Gestion du mode hors ligne (pas accès à l'API)
- Validation côté serveur du format des données et gestion des erreurs
- Appli multi utilisateurs
- Plusieurs numéro par noms et liste d'alias pour chaque contact
- Intégration gestionnaire d'états type [Redux](https://redux.js.org/)
